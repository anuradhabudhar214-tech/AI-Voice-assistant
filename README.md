AI Voice Assistant 

This project is a simple prototype of an AI voice assistant.
It uses text input to simulate speech-to-text and responds
using a small predefined knowledge base.

The assistant avoids incorrect answers by only responding
with known information.

Text-to-speech is implemented using gTTS.
Audio playback is limited to one response to avoid blocking
issues in Google Colab.

To run the project:
- Run the code cell
- Type your query after User:
- Type bye to exit
